Títulos e parágrafos

Títulos

A tag <h> é usada para criar títulos e cabeçalhos em uma página da web. Ela é seguida por um número de 1 a 6, indicando o nível de importância e tamanho do título.

  1. <h1>: É o título de maior importância e tamanho em uma página. Geralmente é usado para o título principal da página ou seção.
  2. <h2>: É um título secundário, um pouco menor que o <h1>. Pode ser usado para dividir o conteúdo em seções principais.
  3. <h3>: Este título é menor do que o <h2> e é geralmente usado para subseções dentro de uma seção <h2>.
  4. <h4>, <h5>, <h6>: São títulos de tamanho progressivamente menor. Eles são usados para subdividir ainda mais o conteúdo, se necessário.

Exemplo:

        
            <h1>Título 1</h1>
            <h2>Título 2/<h2>
            <h3>Título 3</h3>
            <h4>Título 4</h4>
            <h5>Título 5</h5>
            <h6>Título 6</h6>

Resultado:

Título 1

Título 2

Título 3

Título 4

Título 5
Título 6

Parágrafos

A tag <p> é usada para criar um parágrafo de texto. Ela é uma das tags mais básicas e amplamente utilizadas em documentos HTML para estruturar o conteúdo textual. Quando você envolve um trecho de texto com a tag <p>, o navegador interpreta isso como um novo parágrafo, aplicando espaçamento antes e depois do texto para criar um formato de parágrafo.

Exemplo:

        
            <p>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Sequi, magni at veritatis alias laudantium sunt dolore quae, ab, nostrum qui eaque atque cum ratione esse officiis quas aliquam ódio! Modi.</p>

Resultado:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Temporibus quo, praesentium iusto eius et accusantium. Aut rerum deserunt suscipit, natus ipsa quaerat laboriosam exercitationem excepturi ipsam omnis dicta enim molestias!


Os títulos devem ser usados para destacar seções importantes do conteúdo, enquanto os parágrafos são usados para organizar o texto em blocos legíveis.

Formatação de texto

Para formatar textos no HTML, utilize as seguintes tags:

  • <b>: Torna o texto ficar em negrito, mas não carrega um significado especial em termos de semântica.
  • <strong>: Além de tornar o texto em negrito, esta tag também implica que o texto é semanticamente forte ou importante.
  • <i>: Torna o texto em itálico, indicando que ele é um texto com ênfase ou variação no estilo, mas sem alterar o significado.
  • <s>: Risca o texto, indicando que algo foi removido ou descontinuado.

Exemplo:

        
            <p>Esse é um texto <b>negrito genérico</b></p>
            <p>Esse é um texto <strong>negrito relevante</strong></p>
            <p>Esse é um texto <i>itálico</i></p>
            <p>Esse é um texto <s>taxado</s></p>

Resultado:

Esse é um texto negrito genérico

Esse é um texto negrito relevante

Esse é um texto itálico

Esse é um texto taxado

Observação: embora a formatação de texto com o HTML seja possível, não é muito recomendável devido a uma questão estética e além de não ser o uso ideal da tag em si. O HTML é uma linguagem de marcação pensada em semântica, portanto se quiser adicionar estilo, o correto é usar CSS.

Comentários

Um comentário <!—> é uma parte do código que não é executada pelo computador, ou seja, não aparece para o leitor comum, mas serve para adicionar notas e explicar o que está acontecendo no código. Os comentários ajudam os programadores a entender e documentar o código, facilitando a colaboração e a manutenção do projeto.

Exemplo:

        
            <!--meu comentário-->

Atalhos:

  • Ctrl + ; (ou ctrl + K + C): transforma qualquer texto em um comentário automaticamente;

Links externos

Links externos são hiperlinks que direcionam para páginas fora do site atual, permitindo acesso a conteúdo externo.

O atributo href define o URL do link externo. Quando o usuário clica no link, ele será redirecionado para o endereço fornecido.

Exemplo:

        
            <a href="https://linkexterno" target="_blank">Clique aqui</a>
